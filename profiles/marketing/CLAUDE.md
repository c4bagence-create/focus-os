## Frameworks marketing actives

### Pub qui convertit (AIDA)
1. **Hook:** 3 premieres secondes — stopper le scroll, 1 phrase
2. **Probleme:** nommer la douleur exacte du persona
3. **Promesse:** benefice concret (benefice, pas feature)
4. **Preuve:** chiffre, temoignage, avant/apres
5. **CTA:** 1 action, verbe fort, urgence si possible

### Analyser une campagne
1. Quels KPIs ? ROAS > 2x minimum, CTR > 1.5%
2. Quel segment / creatif performe le mieux ?
3. Couper ce qui ne convertit pas sous 72h
4. Scaler ce qui depasse le seuil de rentabilite

### Calendrier contenu social media
1. storm analyse: formats qui performent sur la plateforme cible
2. tasha genere 30 posts/captions en batch
3. tommy cree les templates visuels
4. Programmer via Metricool, Buffer ou Later

## KPIs a surveiller

| Metrique | Seuil critique | Seuil bon |
|----------|---------------|-----------|
| ROAS | < 2x = stop | > 4x = scaler |
| CTR | < 1% = changer creatif | > 2% = bon |
| CVR landing | < 1% = revoir page | > 3% = optimiser |
| CAC | > LTV/3 = danger | < LTV/4 = sain |
| Open rate email | < 20% = revoir objet | > 35% = fort |
| CTR email | < 1.5% = revoir copy | > 4% = fort |

## Checklist campagne

- [ ] Persona defini (age, douleur specifique, comportement achat)
- [ ] Angle unique (pourquoi TOI et pas le concurrent ?)
- [ ] 3 variations copy testees (hook different a chaque fois)
- [ ] Tracking configure (pixel, UTMs, evenements de conversion)
- [ ] Landing en accord avec la promesse de l'ad (message match)
- [ ] Budget test vs scale defini (ex: 30€/j test → 150€/j si ROAS ok)
- [ ] Sequence post-clic ready (landing → email → retargeting)
