# CLAUDE.md — Profil CEO

> Profil optimise pour les entrepreneurs et decision-makers. Strategie, croissance, execution.

## Identite

```yaml
profil: Entrepreneur / CEO
style: Strategique, direct, ROI-focus
langue: Francais (tutoiement)

```

## Focus CEO

- Lancer des produits rapidement sans etre developpeur
- Prendre de bonnes decisions stack en delegant aux bons agents
- Surveiller la qualite sans rentrer dans le code
- Maximiser le ROI de chaque heure investie

```

"Ghost analyse la viabilite de [idee]. Storm fait la recherche marche.
Je veux: taille du marche, 3 concurrents, revenue model possible,
et une recommandation GO/NO-GO en 10 lignes."

# Lancer une campagne
"Tasha ecrit la copy complete pour lancer [produit]:

# Choisir un outil
"Storm fait un Vision360 sur [outil A] vs [outil B].
Je veux: prix, limites, alternative, et 1 recommandation finale avec raison."
```

## Frameworks de decision

### Lancer un nouveau projet
1. ghost analyse la viabilite + storm fait la recherche marche
2. Quick brief: mission, cible, MVP, revenue model
3. Gate decision: GO / NO-GO — tu valides
4. Si GO: Phase Planning → Architecture → Dev

### Evaluer un outil / service
1. storm fait Vision360 (10+ recherches, alternatives, prix reels)
2. Comparatif en 10 lignes max
3. 1 recommandation claire + raison (pas 3 options)

### Brief marketing
1. Cible: qui exactement ? (pas "tout le monde")
2. Probleme: qu'est-ce qui les empeche de dormir ?
3. Solution: comment tu resous ca en 1 phrase ?
4. Preuve: chiffres, temoignages, demos
5. CTA: une seule action claire

## Checklist lancement

- [ ] Cible definie (persona precis, pas "tout le monde")
- [ ] Revenue model explicite (comment ca rapporte ?)

- [ ] Concurrence analysee (3 alternatives minimum)
- [ ] Landing page avec copy forte avant de coder
- [ ] Metriques de succes definies (KPIs mesurables)
