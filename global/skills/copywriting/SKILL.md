## 1. FRAMEWORKS FONDAMENTAUX

### Choisir selon le niveau de conscience du prospect
| Awareness level | Framework | Usage |
|----------------|-----------|-------|
| Pas conscient du probleme | AIDA | Ads froides, contenu educatif |
| Conscient du probleme | PAS | Service pages, landing, ads ciblees |
| Cherche deja une solution | BAB | Sales pages, case studies, temoignages |

**Erreur #1** : utiliser AIDA (pour les inconscients) quand le public est deja solution-aware → utiliser BAB.

### AIDA — Attention, Interest, Desire, Action
- **A** : Hook qui stoppe le scroll (stat choc, question, contradiction)
- **I** : Relier a la vie/business du lecteur
- **D** : Benefices concrets, pas features
- **A** : CTA unique, precis, en gras, positionne ou le regard tombe naturellement

### PAS — Problem, Agitate, Solution
- **P** : Nommer le probleme exactement (le prospect doit se reconnaitre)
- **A** : Aggraver, amplifier le cout de ne PAS resoudre
- **S** : Offrir le chemin de sortie = ton produit

### BAB — Before, After, Bridge
- **Before** : Situation actuelle douloureuse
- **After** : Vision de la transformation
- **Bridge** : Comment tu y arrives (ton offre)

### Hybridation (niveau expert)
PAS pour le hook → FAB (Features, Advantages, Benefits) pour le corps → AIDA pour le close.
C'est comme ca qu'on cree du copy qui convertit sans paraitre generique.

## 2. HOOKS HAUTE CONVERSION

### Types de hooks (social media / ads 2026)
| Type | Exemple | Quand utiliser |
|------|---------|----------------|
| **Pattern interrupt** | Contenu lent et calme dans un feed rapide | Saturation de contenu identique |
| **Open loop** | "Ce que personne ne te dit sur X..." + resolution a la fin | Videos, threads, emails |
| **Curiosity gap specifique** | "J'ai teste 12 outils de scraping. 1 seul a passe Cloudflare." | Audience deja dans le domaine |
| **Stat choc** | "84% des commerciaux citent le prix. Seulement 26% ont raison." | Ads, headers, emails |
| **Contradiction** | "Arrete de travailler plus. Voici pourquoi ca te coute des clients." | Landing pages B2B |

**Regle 2026** : Les hooks generiques ("Vous voulez gagner plus ?") sonnent spam. Les hooks specifiques ("J'ai fait $100K avec cette sequence de 3 emails") convertissent.

**Premier 1-3 secondes** : decisif. Le contenu avec un fort hook genere 1200% de partages supplementaires (source: swiftcopy.io 2026).

## 3. EMAIL SEQUENCES SAAS

### Structure onboarding (activation)
- **J+0** : Email de bienvenue — quick win immediat, 1 seule action
- **J+1** : Feature cle #1 — benefice concret, pas tutorial
- **J+3** : Cas d'usage — story d'un client similaire
- **J+7** : Upgrade trigger — si pas active → offre essai etendu / social proof

### 3 tracks a implementer (source: 3andfour.com)
1. **Quick Win** : Amener a un premier succes en < 5 min
2. **The Hook** : Repeter le loop "Aha!" (garder l'user actif)
3. **Conversion** : PQL-driven (Product Qualified Lead) → upgrade

### Regles CTA
- 1 seul CTA par email
- Formule comme un benefice, pas un verbe vide ("Voir comment doubler X" vs "Cliquer ici")
- Positionne ou le regard tombe naturellement (apres la valeur, jamais avant)
- CTA specifique = +161% de conversions (source: copyhackers.com)

## 4. SOCIAL MEDIA COPY 2026

### Ce qui convertit (source: heyorca.com, socialchamp.com)
- Human-first : authentique > poli
- Respecte le temps de l'audience : court + utile
- Curiosity gap specifique (pas generique)
- Format natif a la plateforme (pas le meme copy Instagram / LinkedIn / TikTok)

### Formule court-format (TikTok, Reels, Shorts)
1. Hook visuel ou texte (0-3s) → Pattern interrupt
2. Promesse de valeur (3-8s)
3. Developpement / proof (8-45s)
4. CTA discret mais clair (fin)

---

## 5. ANTI-PATTERNS

- "Solution innovante" / "de qualite" / "leader" → jargon vide, supprimer
- Plusieurs CTAs dans le meme email → taux de clic divise par 3
- Features avant benefices → toujours inverser
- Copy identique sur tous les canaux → chaque plateforme a son format natif
- Audience awareness ignoree → utiliser le mauvais framework = zero conversion
